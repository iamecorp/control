Решение:
Ввод данных:

В программе предусмотрен ввод массива строк. Пользователь может ввести его с клавиатуры, либо массив может быть задан на старте выполнения алгоритма.
Фильтрация строк:

Для каждой строки в массиве проверяется её длина.
Если длина строки меньше или равна 3 символам, она добавляется в новый массив.
Формирование нового массива:

На основе результатов фильтрации формируется новый массив, содержащий только строки с длиной не более 3 символов.
Вывод результата:

Полученный новый массив строк выводится на экран.
Примеры:
Входные данные: ["Hello", "2", "world", ":-)"]

Результат: ["2", ":-)"]
Входные данные: ["1234", "1567", "-2", "computer science"]

Результат: ["-2"]
Входные данные: ["Russia", "Denmark", "Kazan"]

Результат: []
Использование массивов:
Для решения задачи используются массивы. Отсутствие использования коллекций (например, списков) обусловлено требованием выполнить задачу исключительно с использованием массивов.git
